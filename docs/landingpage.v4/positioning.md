# Dezygn Landing v4 — Positioning Brief

Source framework: April Dunford, *Obviously Awesome*.
Copy framework: Eugene Schwartz, *Breakthrough Advertising*.
Date: 2026-04-07.

---

## 1. Competitive Alternatives (what they'd use if Dezygn didn't exist)

1. **Stitched creative stack** — Midjourney/ChatGPT + Photoshop + Google Drive + email + a feedback doc/Loom. The dominant status quo.
2. **Single-purpose AI product photo tools** — Flair.ai, Pebblely, Booth.ai, Photoroom AI. One shot, then bounce out.
3. **General AI chat with image gen** — ChatGPT, Midjourney, Gemini. No brand memory, no workflow.
4. **Creative review/PM tools** — Frame.io, Filestage, ReviewStudio. Feedback loop only, no generation.
5. **Done-for-you AI photo agencies** — pay someone else.

## 2. Distinct Capabilities

- **Awa**, a conversational design agent with ~15 specialized tools (gen, edit, vision critic, prompt enhancement, brand awareness, web search, Shopify product fetch, video).
- **Visual Syntax v2.0** — structured prompting baked in, the user never writes a prompt.
- **Brand + Campaign memory** — persistent client context across sessions.
- **Recipes** — pre-built ingredient-slot templates for repeatable shots.
- **Ingredients** — products, logos, refs as first-class persistent assets.
- **Built-in client share + structured 5-star feedback** in the same tool.
- **Outreach module** — generate sample creatives on a prospect's actual products + write the cold email. No other creative tool prospects.
- **Concept Review** — internal approval before client send.
- **Video** alongside stills in the same brand/campaign.
- **Built by an operating agency (Fairway Creatives)** — dogfooded daily.
- **Team-based credit pool**, **Quality Mode + Vision Critic**, **concurrent generation**.

## 3. Differentiated Value (themes)

**Wedge — Run the entire client creative loop in one tool, not seven.**
Win the client → produce → get it approved → iterate. AI is finally a production line, not a toy.

Reinforcers:
- **Ship pro-grade AI photography without being a prompt engineer** (Awa + Visual Syntax + Recipes + Vision Critic).
- **Turn AI photography into a sales weapon, not just a delivery tool** (Outreach).

## 4. Best-Fit Accounts

- Freelance designers and 1–10 person creative studios serving DTC / Shopify brands.
- Producing product photography or paid social creative as core deliverable.
- 5–30 active client relationships.
- Currently stitching Midjourney/ChatGPT + Photoshop + Drive + email.
- Growth-mode (want to pitch new clients).
- Buyer = user (owner/operator).

Anti-fit: in-house brand teams, enterprise agencies, prompt-engineer hobbyists.

## 5. Market Category

Existing categories misfire:
- "AI image generator" → triggers Midjourney comparison, hides the loop.
- "Creative project management" → hides generation.
- "Product photography software" → hides outreach + feedback.

**Constructed category:** **AI Creative Studio for DTC Agencies**
(Safer concrete fallback: *AI Photography Studio for Agencies*. Bolder: *AI Creative OS*.)

## Schwartz Layer — Copy Strategy

**Market sophistication:** Stage 4–5. Buyers have seen Midjourney, Flair, Pebblely, every "AI product photo" tool. Mechanisms are exhausted.
→ **Strategy:** identification advertising. Talk *to the prospect*, not about the product. Lead with who they are and what their day looks like.

**State of awareness:** Solution-Aware / Product-Aware. They know AI photography exists. They've tried 2–3 tools. They are tired of the duct tape.
→ **Headline strategy:** name the desire and the *new mechanism* (the loop).

**Mass desire being channeled:** Stop bleeding hours stitching tools together so I can take on more clients without hiring.

**Headline candidates:**
- *The AI creative studio agencies actually run their week from.*
- *Win the client. Make the work. Get it approved. Without leaving one tab.*
- *Your Midjourney + Photoshop + Drive + email stack, collapsed into one studio.*
- *Built by an agency that ships client work on it every day.*

**Sub-head:** For freelance designers and DTC studios who want to pitch, produce, review, and approve — in one place.

**Three core "so what" promises:**
1. **Pitch with finished work in hand.** Outreach generates samples on the prospect's actual products. No tool does this.
2. **Produce without prompt-engineering.** Awa + Visual Syntax write the prompts. You describe the shot.
3. **Close the loop in the same tab.** Share, collect 5-star structured feedback, iterate, approve.

**Proof:** Built and used daily by Fairway Creatives. Bruce replaced a $4,800 shoot in 60 seconds. Milan landed a $2,000 contract.

**Risk reversal:** 50 free credits, 30-day money-back on paid plans.

**Tone:** First person, operator-to-operator. No "we/our/us." Short sentences. No em dashes in headlines. No hype.
